
🎬 VHSFlix - React Movie App

Aplicación web tipo Netflix desarrollada con React que permite explorar películas, ver trailers y gestionar favoritos.

Este proyecto está enfocado en demostrar habilidades reales en desarrollo frontend, consumo de APIs y manejo de estado global.

🛠️ Tecnologías utilizadas
⚛️ React + Vite  
🟨 JavaScript (ES6+)  
🎨 Tailwind CSS  
🧠 Context API + useReducer  
🔀 React Router  
🌐 TMDB API  
✨ Funcionalidades
🔍 Búsqueda de películas en tiempo real
🎭 Navegación por categorías (Acción, Comedia, Terror, Animación…)
🎬 Visualización de trailers
❤️ Sistema de favoritos
🎞️ Scroll horizontal estilo Netflix
📱 Diseño responsive
🧩 Arquitectura
Estado global manejado con Context API y useReducer
Componentes reutilizables
Separación de lógica y presentación
Consumo de API externa (TMDB)
Manejo de rutas con React Router

⚙️ Instalación y uso
Clonar el repositorio:

git clone https://github.com/alexisrrh/proyecto-Peliculas.git

Instalar dependencias:

npm install

Ejecutar el proyecto:

npm run dev

🔑 Configuración

Crear un archivo .env en la raíz del proyecto:

VITE_API_KEY=tu_api_key_aqui
